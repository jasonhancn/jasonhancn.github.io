从零开始的VPS生活 Vol0：如何花掉你的钱

September 02, 2016
Linux运维

屯了超级久的VPS入门系列，基本上就是我自己折腾历程的记录，干货充足。

序言

对于广大码农和准码农群体，有台可以任凭差遣的服务器可谓美事一桩。这个系列是经过数次不满意的配置之后，反复调整出的自用方案，适合有一些基础的新玩家自行配置使用。

P.S. 对写于2016年的最初的两篇内容进行了极大的调整，因为用词太羞耻了 。

确认预算

在玩自己的服务器之前，首先我们要有一台服务器（废话）。所以，第一篇教程就是如何挑选一台心仪的服务器。

想拥有一台真正可以对外提供服务的服务器，目前有三种选择：服务器托管，Bare Matel，还有就是VPS。

服务器托管需要自己运维这台服务器，我们的目的是为了自用，而不是靠这台机器赚钱，所以意义不大。

Bare Matel是云服务商托管的独立实体机，可以像虚拟机一样操作，性能又强，你猜猜缺点是什么？当然是贵喽。

因此，其实我们的预算只能买一台VPS，也就是虚拟机。

既然需求已定，我们可以通过下面这些自我问答，确定一下自己的需求。

  • 学习Linux？ 本机上开一个虚拟机可以满足大部分的需求。

  • 科学上网？ 网上卖的科学上网账号价格更低一些（当然，不是很安全也是一个问题）。

  • 搭建博客？ github page + hexo/jekyll/hugo + gitalk/gitment即可无成本搭建。

以上是一些非刚需的情景。当然，如果满足一二，经济允许，买来学习Linux那是极好的。

  • 我想有一个7x24在线的机器，帮我每天抓取更新的博文/小说/小姐姐照片。

  • 我想搭建个人网盘，随时随地访问我的文件。

  • 我想做个APP，需要接口能够外网访问。

  • （更加个性化的需求）

如果你有以上需求，那么恭喜你，你的花销绝对物有所值。

选择VPS提供商

选择一款对的VPS是成功进入折腾生涯的大前提。通过下面的问答，可以对自己需要的VPS有一个大致的模样。

  • 需要科学上网么？ 是 —— 必选海外VPS。

  • 是学生么？ 是 —— 阿里云、腾讯云等有学生优惠，价格极低。

  • 需要高速/大量上传下载么？ 是 —— 海外VPS的流量，带宽比较便宜。

  • 需要低延迟？ 是 —— 物理距离和较小的海外出口是无法用优化解决的问题，必选国内VPS。

  • 我要肝舰娘！！ —— 必选日本VPS

选定地区之后，就要选择服务商了。

因为我们买VPS的目的是自用，性价比和使用的便捷程度应该是最应该考虑的地方，稳定运行时间有几个9这种问题其实并不重要。

这里以我选择的服务商Vultr为例，说说影响使用体验的一些细节：

  • 付款支持paypal+银联卡，支付宝，门槛低。（不过莫名不支持我的VISA全币卡…）

  • 按小时计费，实时创建、删除，试错成本低。

  • 不支持直接更换主IP，被墙后的恢复成本高。

  • 支持免费的全盘Snapshot备份，间接实现换主IP。

  • 有3.5刀的超廉价选项。

  • 响应ticket的速度很快，客服相对可靠。

  • 因为以上原因，天朝科学上网用户极多，较容易被ban。

查阅一下常见的国外廉价VPS提供商（知名的还有Linode，BudgetVM等等），提供的功能大致相同，可以根据付款方式、价位和天朝用户数量等因素决定究竟选择哪个提供商。

关于海外VPS选择哪个国家的机房，建议直接多创建几台机器测试，哪个快用哪个。更多细节可以用besttrace，看一下所选的VPS到你的电脑是否直连。（比如广东移动的IP到我的日本服务器就要绕香港，联通和电信就是直连）。

如果你需要用日本IP去肝游戏，看NETFLIX/AbemaTV，要注意日本很多IP都是服务商买过去并迁移到日本机房的他国IP，这种情况就需要先搭建简易的代理，测试可以使用目标APP后，再进行下一步。仅仅考察IP地址，无法确认那些内容方会不会把这个IP认为是日本IP。

P.S.如果预算充足，直接选AWS、Azure之流就可以为所欲为了。

在选购的时候，看到OpenVZ这个字眼，尽可能的避而远之（比如著名的bandwagonhost/搬瓦工）…这种虚拟化方式虚拟的不彻底（不过好歹是比docker强的），没法随便换内核导致一些调优的方案无法使用，而且通常超售很严重，会影响连接速度。

关于不需要搭建科学上网的学生党到底选阿里云还是腾讯云，这当然是谁便宜选谁了…摊手~

买个域名

在https越来越普及的当下，域名的重要性逐渐提高了，也越来越便宜了，搞一个域名是非常有必要的。后文有很多用到Let`s Encrypt证书的地方，而签发这个证书是要用一个域名的，所以不要吝啬那几块钱，买一个域名吧（当然了，如果你的朋友有自己的域名，你也可以向他要一个三级域名。但不要用免费域名，不稳定，麻烦很多）。

国内的注册商目前注册一切域名用于任何区域的服务器都需要实名登记，所以如果不是国内服务器的话建议选择海外注册商（当然，国内的注册商配国内服务器还是很省心的，国内的服务商会帮你搞备案的材料）。

海外注册商比如Godaddy（最近差评如潮），name.com，namesilo等，主要注意一下续费价格（经常会出现第一年注册很便宜，续费很贵的情况），有没有免费的whois保护等等。

不考虑SEO的情况，域名什么结尾都行，所以那几个0.99刀一年的域名不失于一个好选择。

结语

总之，如果你能找着上面这些内容选择一套心仪的购买方案，那我们就可以进入下一环节，开启折腾之旅了。
