彻底删除Wine

2016-07-15 极客生活

在半年前的某一天，脑子不知道哪根弦抽了，想给Linux上装个115浏览器下东西，然后就想到了Wine。

说道做到，sudo apt -install wine（Xubuntu 16.04，apt-get推荐缩写成apt，apt的界面也更好看点）。

软件源里的Wine是1.6，安装115浏览器，失败，boom。

然后搜一下，说是Wine版本越新越好用，果断官网走起，1.8.4。

苍天啊大地啊装上了，打开，boom。

玛德智障，一个chrominum到底怎么做成这鬼样的啊…

怒删之，sudo apt remove wine。

过了好久，Whisker Menu里一直有Wine的各种残留，打开方式也是…强迫症伤不起啊…

嗯，再搜一下。

Google大法打了我一巴掌，脑补一句台词“谁教你这么卸东西了？”

显然应该这样：

sudo apt purge wine*
sudo apt autoclean
sudo apt autoremove

remove会把配置文件留下，然而我不想再见到它了，应该用purge跟它say goodbye。（用了remove的可以用新立得来清除残留）

wine*会删掉跟软件包名带wine的软件，不过注意有longene QQ的别误删了。

autoclean和autoremove两兄弟就不用解释了，干啥都得敲一下。

菜单项用菜单编辑器删掉。

齐活！

所以说，我玩linux还是图样图森破，不断学习吧。
